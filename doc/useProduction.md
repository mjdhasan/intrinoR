### `useProduction`
***
***

### Description

 When the api calls are directed to the sandbox server there are a limited number of symbols, which return data.
 These include the dow30, "AAPL", etc.

### Usage
```r
useProduction()
```

### Details

 Accessing the production data requires a paid subscription
