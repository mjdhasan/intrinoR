### `getSecurityById`
***
***

### Description

 Returns the Security with the given `identifier`

### Usage
```r
getSecurityById(identifier)
```

### Arguments
Argument      |Description
------------- |----------------
```identifier```     |     String A Security identifier (Ticker, FIGI, ISIN, CUSIP, Intrinio ID)
### Value

 df tibble
