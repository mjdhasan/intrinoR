### `getFundamentalById`
***
***

### Description

 Returns detailed fundamental financial data for the given `id` .

### Usage
```r
getFundamentalById(id)
```

### Arguments
Argument      |Description
------------- |----------------
```id```     |     String The Intrinio ID for the Fundamental
