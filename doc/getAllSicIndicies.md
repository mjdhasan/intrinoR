### `getAllSicIndicies`
***
***

### Description

 Return SIC Indices Meta-Data

### Usage
```r
getAllSicIndicies(opts = list())
```

### Arguments
Argument      |Description
------------- |----------------
```opts```     |     named list Optional query parameters
```opts$page_size```     |     the number of results to return (defaults to 100)
